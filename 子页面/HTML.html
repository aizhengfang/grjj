初次接触 HTML 时，密密麻麻的标签代码让我望而生畏。但在老师 “HTML 就像搭建积木” 的比喻下，我开始尝试用div搭建网页框架， 用img插入图片，每完成一个简单页面，都充满成就感。后来，我通过制作个人简历网页，将header、nav等语义化标签融入其中，不仅 代码结构更清晰，也让我理解了 HTML 在 SEO 优化中的作用。​学习过程中，我也遇到不少挑战。比如，ul和ol列表标签的嵌套使用， 常常导致页面布局混乱；table表格的tr、td元素，在复杂表格设计时容易出错。我通过反复阅读 W3C 标准文档，观看教学视频，结合实际案例练习，逐步攻克这些难题。​ 现在，我深刻体会到 HTML 是网页开发的基石。它看似简单，实则蕴含着对网页结构的深度理解。未来，我会继续将 HTML 与 CSS、JavaScript 结合，创造出更美观、交互性更强的网页。
