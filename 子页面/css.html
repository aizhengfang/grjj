刚开始学习 CSS 时，我以为只是简单地给 HTML 元素添加颜色和字体。当我尝试用div搭建一个简单网页，为其添加样式时， 才发现问题重重。多个选择器同时作用于一个元素，样式优先级混乱，页面效果与预期大相径庭。比如设置段落文本颜色， 内联样式覆盖了外部样式表的设置，让我意识到 CSS 层叠规则的重要性。布局学习更是一道难关。传统的float布局， 元素脱离文档流后，常导致父元素高度塌陷，我花费大量时间调试。后来接触到Flex布局，仿佛打开新世界大门。 通过实际项目，我用Flex实现页面布局，不仅提高了开发效率，页面响应式效果也更好。​ 我深刻体会到 CSS 是网页设计的灵魂，它赋予网页独特风格与良好用户体验。未来，我会继续深入探索 CSS 新特性，让网页设计更上一层楼。
